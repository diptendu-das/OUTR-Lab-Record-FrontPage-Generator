# 🎓 OUTR Lab Record Cover Page Generator

A sleek, modern, and lightning-fast web application built with **React** & **Tailwind CSS** that instantly generates strictly formatted, university-compliant "Lab Record Cover Pages" for students of Odisha University of Technology and Research (OUTR). 

Forget wrestling with MS Word templates! Just plug in your academic details and grab a perfectly sized A4 PDF in a single click.

## ✨ Features

- **Blazing Fast**: Single-page application rendering, no waiting.
- **University Standard Formatting**: The output preview rigorously maintains the exact formatting parameters (red double borders, specific alignments) required by the OUTR administration.
- **Modern User Interface**: A beautifully crafted, responsive input form featuring glassmorphism elements, intuitive color pickers, and smooth interactions.
- **Zero-Friction PDF Exports**: Uses `html-to-image` and `jsPDF` to instantly convert the DOM node into a crisp, high-resolution A4 `.pdf` file. 
- **Granular Customization Features**:
  - Dynamically adjustable heading and UI colors.
  - Optional toggle to include/exclude "Submitted To" information.
  - Optional stylized wrapping boxes for submission data.
- **Interactive Modals**: Clean and accessible pop-up modules for About, Help, and Contact info.

## 🚀 Quick Start (Local Development)

To run this application locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/diptendu-das/OUTR-Lab-Record-FrontPage-Generator.git
   ```
   
2. **Navigate to the project directory:**
   ```bash
   cd OUTR-Lab-Record-FrontPage-Generator
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the Vite development server:**
   ```bash
   npm run dev
   ```
   *The app will be running locally at `http://localhost:5173`.*
